On the RaspberryPi
==========================
see http://www.rabbitmq.com/install-debian.html

Add the following line to your /etc/apt/sources.list:

deb http://www.rabbitmq.com/debian/ testing main

Run apt-get update.

sudo apt-get install rabbitmq-server
